# Luta Imobiliária

> Jogo de RPG para terminal Linux com ASCII art, ambientado na crise imobiliária de São Paulo — primeiro contato com Swift.

## Sobre o Projeto

**Luta Imobiliária** é um jogo de RPG rodado no terminal, desenvolvido em grupo com [@dayoleal](https://github.com/dayoleal) e [@7uigi](https://github.com/7uigi) como primeiro contato com a linguagem Swift.

A inspiração veio dos clássicos RPGs dos anos 2000, e toda a composição visual foi construída com **ASCII art**. A narrativa tem como pano de fundo a crise imobiliária de São Paulo, levando o jogador a enfrentar puzzles e combates dentro desse universo.

## Funcionalidades

- Tela inicial com menu de opções (jogar, ajuda e sair)
- Menu de ajuda com mecânicas e dicas do jogo
- Prólogo narrativo contextualizando a história
- Puzzles com objetos cenários interativos
- Combates com adversários secundários e chefão final
- Interface totalmente em ASCII art

## Como Rodar

1. Clone o repositório:
```bash
git clone https://github.com/LosadaT/Luta_Imobiliaria.git
```

2. Navegue até a pasta do projeto:
```bash
cd Luta_Imobiliaria
```

3. Compile e execute com Swift:
```bash
swift run
```

> Requisito: Swift 5.5+ instalado em um ambiente Linux ou macOS com terminal.
